# 卢沟桥数字文博专题页面 - 技术规格说明书 (SPEC.md)

## 1. 项目概述
*   **项目名称**: 桥韵中华 - 卢沟桥数字孪生体验
*   **项目类型**: 交互式专题网页 (Single Page Application)
*   **核心目标**: 通过 3D 交互、动态 Shader 和数据可视化，展示卢沟桥的历史、建筑美学及抗战历史，冲击评比奖项。
*   **技术栈**: HTML5, CSS3, JavaScript (ES6+), Three.js (3D), GSAP (Animation).

## 2. UI/UX 规格

### 布局结构
*   **导航栏 (Nav)**: 固定顶部，透明渐变背景，含有模块锚点（历史、3D展示、建筑、诗词）。
*   **Hero Section (首屏)**:
    *   背景: WebGL Shader 实现的“卢沟晓月”动态水面波纹效果。
    *   主体: 3D 悬浮的卢沟桥模型（使用几何体堆叠模拟或加载模型）。
    *   文字: 竖排排版的标题，配合打字机效果显现。
*   **第一章节: 历史的跨度 (History)**:
    *   形式: 横向滚动的 时间轴 (Timeline)。
    *   交互: 滚动触发年份卡片展开。
*   **第二章节: 建筑巡礼 (Architecture)**:
    *   形式: 3D 场景切换，用户可点击桥体不同部位（桥墩、栏板、石狮）弹出详情 Modal。
*   **第三章节: 红色记忆 (War & Peace)**:
    *   形式: 黑白照片与彩色现实的对比滑动 (Before/After Slider)。
    *   背景: 略带噪点的复古滤镜。
*   **页脚**: 鸣谢、资料来源。

### 视觉设计
*   **配色方案**:
    *   主色: ` #1a1a1a` (深墨色 - 背景/夜空)
    *   辅色: `#c5a059` (金色 - 标题/高亮)
    *   点缀: `#8b0000` (暗红色 - 抗战章节强调)
    *   文字: `#e0e0e0` (月白色)
*   **字体**:
    *   标题: `ZCOOL XiaoWei` (站酷小薇体) - 古典韵味。
    *   正文: `Noto Serif SC` (思源宋体) - 易读且具书卷气。
*   **动效**:
    *   全局: 页面滚动平滑 (Locomotive Scroll 风格或自定义)。
    *   3D: 镜头缓动 (Damping)，材质反光。

## 3. 功能规格

### 核心功能
1.  **WebGL 动态背景**:
    *   实现一个 Fragment Shader，模拟月光倒影在水面上的效果。
    *   鼠标移动可轻微扰动水面纹理。
2.  **3D 交互展示**:
    *   使用 Three.js 渲染场景。
    *   加载卢沟桥模型（若无可用模型，使用程序化生成的石拱桥几何体替代：多个半圆拱形 Mesh + 平面桥面）。
    *   实现 Raycaster 射线检测，鼠标悬停桥体部件时高亮。
    *   点击部件，显示详情卡片。
3.  **GSAP 动画**:
    *   页面元素进场动画 (ScrollTrigger)。
    *   标题文字的交错动画。

### 数据处理
*   历史事件数据、桥梁结构数据硬编码在 JSON 对象中，动态渲染到页面。

## 4. 验收标准
*   [ ] 首屏加载无白屏，Shader 背景流畅运行。
*   [ ] 3D 模型可自由旋转、缩放。
*   [ ] 点击石狮/桥墩能正确触发交互事件。
*   [ ] 页面在不同分辨率（PC 1920x1080, Mobile 375x667）下布局正常。
*   [ ] 无控制台关键性报错。
