# 卢沟桥专题页面 - 功能亮点说明 (用于参赛/汇报)

## 一、核心技术亮点 (Technical Highlights)

本项目摒弃了传统的“图文堆砌”式网页，采用 **Web 3D** 和 **交互设计** 相结合的方式，旨在打造一个“数字文博”级别的沉浸式体验。

### 1. 3D 数字孪生交互系统 (Three.js Digital Twin)
*   **程序化建模**: 在没有 3D 美术素材的情况下，使用 Three.js 代码（`TorusGeometry`, `BoxGeometry`）程序化构建了卢沟桥的“十一孔联拱”几何模型。
*   **沉浸式漫游**: 用户可以通过鼠标拖拽，360 度旋转查看桥体结构。
*   **热点交互 (Hotspots)**:
    *   **射线检测**: 利用 Raycaster 技术，鼠标悬停在“石狮”、“桥墩”、“桥面”上时，模型会自动高亮（发光）。
    *   **数据驱动**: 点击模型会弹出详情卡片，实现了“模型即数据”的交互逻辑。

### 2. WebGL Shader 动态意境背景
*   **写在代码里的艺术**: 背景不是静态图片，而是使用 GLSL 着色器语言编写的实时渲染画面。
*   **卢沟晓月**: 
    *   **动态月球**: 实时计算的月球光晕和星点。
    *   **交互式水波**: 鼠标移动会微调月相和光照角度。
    *   **物理反射**: 实现了基于噪声算法的水面波光粼粼效果，模拟月光倒影。
*   **性能优势**: 相比加载 4K 高清背景图，Shader 代码体积极小（< 10KB），且在不同屏幕分辨率下自动清晰度无损。

### 3. 专业的动画引擎 (GSAP)
*   **行业标准**: 引入了 GreenSock 动画平台 (GSAP)，这是业界公认的 Web 动画标准（比 CSS Animation 强大得多）。
*   **摄像机运镜**: 点击右侧“桥墩”、“石狮”按钮时，摄像机会平滑地飞向目标位置，实现了类似电影的“运镜”效果。
*   **平滑导航**: 顶部导航栏点击后，页面会平滑滚动至对应章节，而不是生硬地跳转。

## 二、创新功能模块 (Functional Modules)

### 1. 历史的跨度 (高精度时间轴)
*   **ScrollTrigger**: 利用滚动条深度触发时间轴节点的展开动画，增强阅读节奏感。
*   **动态进度条**: 页面滚动时，垂直时间轴线条会实时填充颜色，指引阅读进度。
*   **交互高亮**: 当前阅读的时间节点会自动高亮放大，配合优雅的入场动画。
*   **内容深化**: 每个历史节点都配有详细的图文描述，而非简单的罗列。

### 2. 诗词歌赋 (Poetry)
*   **沉浸阅读**: 专门设计的竖排排版模块，配合古典字体，展示关于卢沟桥的诗词。

### 3. 流动的史诗 —— 京西门户数据叙事 (Traffic Hub)
*   **双图层可视化**: 底层为 Canvas 绘制的“京畿古地图”（包含永定河走势、京城/西山/天津方位），上层为动态交通流。
*   **ECharts 数据联动**:
    *   折线图展示明清以来“卢沟桥关税”趋势（基于史料模拟）。
    *   **交互同步**: 拖动时间轴时，图表与地图流量的光点密度实时联动。
*   **交互式时间轴**:
    *   模拟 1441-1911 年间的兴衰。
    *   **关键节点标记**: 设有“设关”、“甲申”、“盛世”、“事变”等交互节点，点击即可跳转。
    *   **事件触发**: 当滑块经过关键年份时，右侧会自动弹出历史背景卡片。
    *   **动态反馈**: 时间轴上的进度条颜色会随朝代兴衰变化（清代鼎盛为金色，晚期衰落为灰色）。

### 4. 红色记忆 (影像对比)
*   **Clip-path Slider**: 实现了“历史”与“现实”的横向拖动对比效果，寓意“勿忘历史，珍爱和平”。

### 3. 响应式与无障碍
*   **自适应**: 完美适配 PC 端和移动端（手机端会自动隐藏复杂导航，优化布局）。

## 三、技术栈总结
*   **HTML5/CSS3**: 语义化标签，CSS变量定义高级配色。
*   **JavaScript (ES6+)**: 原生 JS，无 jQuery 依赖。
*   **Three.js**: WebGL 3D 引擎。
*   **GSAP**: 动画与交互控制。
